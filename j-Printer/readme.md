## j-Printer

The component can print a HTML content.

- singleton
- easy usage

__Configuration__:

- `delay {Number}` a delay for evaluating of printer (default: `500`)

__Methods__:

```js
SETTER('printer/print', '<h1>Total.js Platform</h1>');
````
