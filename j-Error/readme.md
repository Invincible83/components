## j-Error

- easy usage
- an `error source` must be `array` like this `[{ error: 'Error Message 1' }, { error: 'Error Message 2' }]`
- if the `error source` is empty or nullable then the component hides element

__Configuration__:
- `icon {String}` an error icon (default: `warning`)
